# Part 3 — Streamlit Dashboard (Titanic)

## How to Run
```bash
cd part3-dashboard
pip install -r requirements.txt
streamlit run app.py
```

Open the provided local URL.

## Features
- Filters:
  - Sex
  - Pclass (class)
  - Age range
- Charts:
  - Survival rate by Sex
  - Survival rate by Pclass
  - Distribution of age (hist)
  - Fare distribution (box)

## Diagnostic Questions Answered
- Who has higher survival rate: men vs women?
- Which passenger class has better survival rate?
- What is the age and fare distribution among survivors vs non-survivors?
